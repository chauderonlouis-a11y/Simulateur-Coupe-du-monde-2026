Como funciona o simulador?

Esta página explica o princípio do site, as regras usadas para as classificações, os melhores 3.ºs e a fase a eliminar.

🚀 Ir diretamente para o simulador
Apresentação

O que o simulador permite

🏆 A ferramenta para simular quem vai levantar o troféu!

  • ordenar as equipas em cada grupo em modo clássico (arrastar e largar);
  • selecionar ou deixar calcular automaticamente os 8 melhores 3.ºs;
  • gerar um quadro de 32 equipas para a fase a eliminar;
  • mudar para o modo especialista para prever os resultados exatos de todos os jogos da fase de grupos e preencher a fase a eliminar na versão “com marcadores”;
  • simular cada ronda a eliminar até ao título.
Modo clássico

Classificação simples dos grupos

No modo clássico, reordenas as equipas de cada grupo por arrastar e largar:

  • os 12 primeiros de cada grupo apuram-se;
  • os 12 segundos também;
  • depois selecionas manualmente os 8 terceiros apurados.

Em seguida, é gerado o quadro de 32 equipas, e podes clicar em cada equipa para escolher a vencedora de cada jogo.

Modo especialista

Prever todos os marcadores

No modo especialista, introduces os marcadores de todos os jogos da fase de grupos. O site calcula automaticamente para cada equipa:

  • jogos disputados;
  • vitórias, empates, derrotas;
  • golos marcados / sofridos e diferença de golos;
  • pontos na classificação.

A classificação de cada grupo é atualizada em tempo real com base nos resultados que introduzes.

Melhores 3.ºs

Como são escolhidos os 8 melhores 3.ºs?

No modo especialista, quando todos os marcadores estiverem preenchidos, as 12 equipas que terminam em 3.º lugar são ordenadas automaticamente de acordo com:

  • pontos;
  • diferença de golos;
  • golos marcados;

Se várias equipas continuarem completamente empatadas, o simulador mostra uma pequena janela para as desempatar pelo fair-play (ordenas da mais para a menos fair-play).

As 8 primeiras deste quadro apuram-se para a fase a eliminar.

Playoffs

Gestão dos vencedores dos playoffs

Algumas vagas do Mundial ainda são desconhecidas (formatos “Vencedor do playoff UEFA …”, “Vencedor do playoff interconfederação …”).

Podes abrir a janela “Escolher vencedores dos playoffs” e substituir estes placeholders pelas equipas que imaginas apuradas (Itália, Ucrânia, Suécia, etc.).

O nome e a bandeira são atualizados em todo o lado:

  • nos grupos (modo clássico);
  • nos jogos do modo especialista;
  • no quadro da fase a eliminar.
Nota

Limites e nota importante

Este simulador é uma ferramenta não oficial, criada por diversão e para facilitar os prognósticos entre amigos.

Tenta ser o mais fiel possível (sorteios, critérios de desempate extremos, etc.), mas se quiseres ter 100% de certeza, consulta o site oficial para ver todas as regras.
