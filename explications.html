Comment fonctionne le simulateur ?

Cette page explique le principe du site, les règles utilisées pour les classements, les meilleurs 3ᵉ et la phase finale.

Présentation

Ce que permet le simulateur

Le site te permet de simuler toute la Coupe du Monde 2026, du tirage des poules jusqu’au champion final :

  • classer les équipes dans chaque groupe (mode classique) ;
  • choisir les 8 meilleurs 3ᵉ qualifiés ;
  • remplir automatiquement un tableau à 32 équipes pour la phase finale ;
  • passer en mode expert pour pronostiquer les scores précis de tous les matchs de poule ;
  • simuler chaque tour à élimination directe jusqu’au titre.
Mode classique

Classement simple des poules

En mode classique, tu réordonnes les équipes de chaque groupe par glisser-déposer :

  • les 12 premiers de groupe sont qualifiés ;
  • les 12 deuxièmes aussi ;
  • tu sélectionnes ensuite manuellement les 8 troisièmes qualifiés.

Le tableau à 32 équipes est alors généré, et tu peux cliquer sur chaque équipe pour désigner la gagnante de chaque match.

Mode expert

Pronostiquer tous les scores

En mode expert, tu entres les scores de tous les matchs de poule. Le site calcule automatiquement pour chaque équipe :

  • matchs joués ;
  • victoires, nuls, défaites ;
  • buts marqués / encaissés et différence de buts ;
  • points au classement.

Le classement de chaque groupe est mis à jour en direct en fonction des résultats que tu saisis.

Meilleurs 3ᵉ

Comment sont choisis les 8 meilleurs 3ᵉ ?

En mode expert, une fois tous les scores remplis, les 12 équipes classées 3ᵉ sont triées selon :

  • les points ;
  • la différence de buts ;
  • les buts marqués ;
  • puis l’ordre alphabétique si tout est encore égal.

Les 8 premiers de ce tableau sont qualifiés pour la phase finale.

Barrages

Gestion des barragistes

Certaines places de la Coupe du Monde sont encore inconnues (formats “Vainqueur barrage UEFA …”, “Vainqueur barrage inter-conf …”).

Tu peux ouvrir la fenêtre “Choisir les barragistes” et remplacer ces placeholders par les équipes que tu imagines qualifiées (Italie, Ukraine, Suède, etc.).

Le nom et le drapeau sont alors mis à jour partout :

  • dans les poules (mode classique) ;
  • dans les matchs du mode expert ;
  • dans le tableau de la phase finale.
Remarque

Limites & note importante

Ce simulateur est un outil non officiel, créé pour le fun et pour faciliter les pronostics entre amis.

Il ne prétend pas reproduire à 100 % toutes les subtilités du règlement officiel (tirages, critères de départage extrêmes, etc.), mais il respecte les grands principes de classement utilisés habituellement.
